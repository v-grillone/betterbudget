# betterbudget

A minimalist budgeting app. Track daily transactions across needs, wants, and investing categories against your monthly budget.

## Stack
- Next.js (App Router) · TypeScript · Tailwind CSS
- Supabase (auth + DB) · Stripe · shadcn/ui · Motion

## Getting Started

1. Copy `.env.example` to `.env` and fill in Supabase + Stripe keys
2. `npm install`
3. `npm run dev` → http://localhost:3000
